## Fibonacci GCD

### Problem Statement

Let $F_n$ be the $n$-th Fibonacci number with $F_1 = 1, F_2 = 1$.

Find $\gcd(F_{24}, F_{18})$.

### Notes

- Use the property: $\gcd(F_m, F_n) = F_{\gcd(m,n)}$
- First find $\gcd(24, 18) = 6$
- Then calculate $F_6$
- Submit the exact value

### Answer

```
8
```
